# CareerFlow AI Landing Page

A modern, animated landing page for CareerFlow AI - The All-in-One AI Companion for Your Career Journey.

## Features

- **Modern Design**: Dark mode with neon gradients and glassmorphism effects
- **Smooth Animations**: Powered by Framer Motion with scroll-based animations
- **Responsive Layout**: Works on all device sizes
- **Interactive Elements**: Hover effects and micro-interactions
- **Performance Optimized**: Lightweight implementation

## Tech Stack

- React.js with TypeScript
- Tailwind CSS for styling
- Framer Motion for animations
- Vite for fast development

## Sections Included

1. **Hero Section** - App name, tagline, and CTAs with animated entrance
2. **Problem Section** - Career challenges with animated cards
3. **Solution Section** - CareerFlow AI modules with hover animations
4. **How It Works** - 3-step process with timeline animation
5. **Technology Section** - Tech stack badges with hover effects
6. **Final CTA** - Strong conversion-focused section

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Customization

- Colors can be adjusted in `tailwind.config.js`
- Animations are controlled through Framer Motion props
- Content can be easily modified in the `LandingPage.tsx` component

## Deployment

The landing page can be deployed to any static hosting service like Vercel, Netlify, or GitHub Pages.
